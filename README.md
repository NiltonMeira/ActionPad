# projetoX
Projeto criado com a ideia de ser um gerenciador de tarefas inovador, diferente de tudo que ja existe no mercado.

Features:

- Login de usuarios
- Acesso a várias ferramentas de anotações
- Bloco de notas com editor de texto
- Checklist 
- Cards task
- Desenho whitebord 
- Tabelas

Design: 
- Tema minimalista 
- Cores neutras
